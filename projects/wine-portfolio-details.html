Project Overview

Introduction: This project delves into understanding the dynamics of Californian wine prices, aiming to predict and classify their fluctuations. Leveraging a diverse set of classification algorithms, the project explores patterns in the wine dataset to ascertain benchmarks for future price trends. By employing machine learning models, the goal is to provide insights into the demand and pricing variations within the Californian wine market.

Dataset Overview: The dataset used for this project is focused on Californian wines, providing valuable information to analyze and predict price changes. Key features, including Year, CountyCode, HarvestedAcres, Yield, and Production, are utilized to comprehend the factors influencing price variations. The target variable, Price Categories, is derived from the Price column, classified into labels 0 (Low), 1 (Medium), and 2 (High). Data preprocessing involves handling null values, and Min-Max Scaling transforms features to ensure a standardized input for the machine learning models.

Purpose: The primary objective is to build predictive models that can classify Californian wine prices into different categories, namely Low, Medium, and High. By employing a variety of classification algorithms, the project seeks to identify the most effective model for predicting price fluctuations. Insights gained from the project will aid in strategic decision-making within the wine industry, providing valuable information about the demand and pricing trends.

Approach:The project adopts a comprehensive approach by utilizing various classification algorithms, including Random Forest, Logistic Regression, Support Vector Classification, K Nearest Neighbors, AdaBoost, Decision Tree, Naive Bayes, Artificial Neural Networks, XGBoost, and Gradient Boosting. The choice of these algorithms allows for a thorough exploration of different modeling techniques. The dataset undergoes preprocessing, involving the removal of null values and the transformation of the target variable into categorical labels. The Min-Max Scaler ensures standardized features for effective model training. By employing grid search and randomized search techniques, the project fine-tunes hyperparameters to optimize model performance.
